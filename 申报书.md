# MoonDataDrift 参赛申报书

项目名称：MoonDataDrift  
项目标识：`moondatadrift`  
项目简介：MoonDataDrift 是一个基于 MoonBit 的数据漂移检测工具，用于比较训练集与新数据的分布变化，面向 ML 监控与数据质量检查场景。

## 项目说明

本项目聚焦一个成熟且可扩展的交叉方向：表格数据漂移检测。当前版本支持在内存表格上完成基线与当前数据的比较，并输出 Markdown 报告，覆盖缺失率、数值列分位数、PSI、KS-like 距离以及类别分布变化。它的设计目标不是做一个炫技型 demo，而是做一个能继续扩展到 CSV 读取、CI 监控、告警规则和可视化报告的实用工具。

## 主要功能

- 自动识别数值列与类别列
- 比较缺失率变化
- 输出数值列 p10 / p50 / p90
- 计算 PSI 与 KS-like 统计
- 输出类别频率漂移摘要
- CLI 直接生成 Markdown 报告

## 实现特点

- MoonBit 为主要实现语言
- 核心逻辑与命令行入口分离
- 提供单元测试与可运行示例
