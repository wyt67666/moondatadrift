# MoonDataDrift 参赛申报书

项目名称：MoonDataDrift  
项目标识：`moondatadrift`  
项目简介：MoonDataDrift 是一个用 MoonBit 实现的数据漂移检测工具，用于比较训练集与新数据之间的分布变化，支持数值列、类别列、缺失率、分位数、PSI 和 KS-like 统计，并通过 CLI 输出简洁的 Markdown 报告。

## 项目说明

本项目聚焦 ML 监控中的“数据漂移”这一成熟交叉方向，当前实现以内存表格对比为核心，已经具备可运行命令、示例数据、单元测试和提交历史，方便后续继续扩展到 CSV 读取、阈值告警、可视化报表和 CI 集成。

## 主要功能

- 自动识别数值列与类别列
- 比较缺失率变化
- 输出数值列 p10 / p50 / p90
- 计算 PSI 与 KS-like 距离
- 汇总类别频率偏移
- 生成可直接提交的 Markdown 报告
