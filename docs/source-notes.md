# Source Notes

MoonDataDrift is an original MoonBit project.

## Inspiration

- ML monitoring and data quality checks
- standard drift metrics such as PSI and KS-style distance
- tabular-data libraries in the MoonBit ecosystem

## Non-goals

- no copied source from private or closed projects
- no vendor-specific integrations in the first version
- no heavy dashboard framework in the first version

## Planned growth

- CSV file loading
- richer report formatting
- trend comparison across multiple snapshots
- GitHub Actions CI for regression tests
